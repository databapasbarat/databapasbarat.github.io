# **App Name**: NIK Check

## Core Features:

- NIK Input: Input field for NIK with validation
- API Integration: Fetches data from `https://api.siputzx.my.id/api/tools/nik-checker?nik=[NIK]`
- Data Display: Displays fetched NIK data in a table format.
- AI Summary: Generates a summary of the displayed NIK data using a generative AI tool to highlight key information.

## Style Guidelines:

- Primary color: Deep indigo (#3F51B5) for a trustworthy and official feel.
- Background color: Light gray (#F5F5F5), close to white, to ensure good readability and a clean layout.
- Accent color: Teal (#009688) to highlight interactive elements and important data.
- Body and headline font: 'Inter' (sans-serif) for a modern, objective, neutral look.
- Simple, clean layout with a focus on readability and data presentation. Use of tables for structured data.
- Use government-themed icons for a more official feel.
- Subtle loading animations when fetching data from the API.
